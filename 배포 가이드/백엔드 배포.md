# 백엔드 배포 가이드

## 사전 조건

- EC2: 기존 BeautyBook 운영 인스턴스
- Java: JDK 21
- PostgreSQL: Docker 컨테이너 `english-agent-hub-postgres`
- 백엔드 포트: `4301`
- systemd 서비스명: `englishagenthub.service`

## 로컬 빌드

```bash
cd /Users/terecal/english-agent-hub-container/english-agent-hub-server
./gradlew build -x test
ls -lh build/libs/
```

배포 JAR:

```text
build/libs/english-agent-hub-server-0.0.1-SNAPSHOT.jar
```

`*-plain.jar`는 실행 JAR가 아니므로 배포하지 않는다.

## EC2 디렉터리 준비

```bash
ssh -i "배포 가이드/hibot-d-server-key.pem" ubuntu@13.209.195.64

mkdir -p ~/english-agent-hub
cd ~/english-agent-hub
```

## PostgreSQL 실행

EC2에서 실행한다.

```bash
cat > ~/english-agent-hub/docker-compose.yml << 'EOF'
services:
  postgres:
    image: postgres:15-alpine
    container_name: english-agent-hub-postgres
    restart: unless-stopped
    environment:
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: postgres
      POSTGRES_DB: english_agent_hub
    ports:
      - "5436:5432"
    volumes:
      - english_agent_hub_postgres_data:/var/lib/postgresql/data
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U postgres"]
      interval: 10s
      timeout: 5s
      retries: 5

volumes:
  english_agent_hub_postgres_data:
    driver: local
EOF

docker compose up -d postgres
docker ps | grep english-agent-hub-postgres
```

## 환경변수 배치

로컬의 `배포 가이드/.env.prod.example`을 참고해 EC2에 `/home/ubuntu/english-agent-hub/.env`를 만든다.

```bash
nano ~/english-agent-hub/.env
chmod 600 ~/english-agent-hub/.env
```

필수 값:

```env
JWT_SECRET=...
API_KEY_SECRET=...
OPENAI_API_KEY=...
ALLOWED_ORIGIN=https://english.dxline-tallent.com
```

## systemd 서비스 생성

EC2에서 실행한다.

```bash
sudo tee /etc/systemd/system/englishagenthub.service << 'EOF'
[Unit]
Description=EnglishAgentHub Backend
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/home/ubuntu/english-agent-hub
EnvironmentFile=/home/ubuntu/english-agent-hub/.env
ExecStart=/usr/bin/java -jar /home/ubuntu/english-agent-hub/app.jar --spring.profiles.active=prod
Restart=on-failure
RestartSec=10
StandardOutput=append:/home/ubuntu/english-agent-hub/spring-boot.log
StandardError=append:/home/ubuntu/english-agent-hub/spring-boot.log

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl daemon-reload
sudo systemctl enable englishagenthub
sudo systemctl start englishagenthub
```

## DB 초기화

운영 데이터가 테스트 데이터일 때만 사용한다.

```bash
docker exec english-agent-hub-postgres psql -U postgres -d english_agent_hub \
  -c 'DROP SCHEMA public CASCADE; CREATE SCHEMA public;'

sudo systemctl restart englishagenthub
```
